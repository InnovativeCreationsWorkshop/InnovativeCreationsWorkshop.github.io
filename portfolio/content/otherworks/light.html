Interactive Light Experience

An Installation

Technology Used: TouchDesigner, Microsoft Kinect

I sought out to create an immersive experience that users could experience using an item that almost everyone carries, a smartphone/cellphone. For this piece, I was inspired by Xinmax's tutorial on flashlight painting in 3D and is very much the backbone of this piece. However, I wanted to add an audio reactive element to it as well.

Examples

This video introduces the nodal network in TouchDesigner, showcasing all the key components behind this interactive experience.

At its core, the project uses light tracing. A webcam detects bright light from a phone screen, and a series of nodes track its movement. A geometry node then brings the trace to life using a line sprite material.

A second geometry node is linked to the trace, generating particles that gradually dissipate as the line moves. Inside this node, an intricate audio processing system is set up.

Process

Audio input is resampled and multiplied to extract frequency values, which are then mapped to the particle system through Drag and Turbulence parameters. This makes the particles dynamically respond to sound. A secondary layer of particles, using a similar technique, enhances the atmospheric and audio-reactive elements. All these nodal networks are then composited to create the final immersive experience—where both user interaction and audio influence the visuals.
